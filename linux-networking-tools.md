# Linux Networking Tools

## ping

Checks connectivity using ICMP.

Example:
ping google.com

---

# traceroute / tracert

Shows packet path across networks.

Example:
traceroute google.com

Windows:
tracert google.com

---

# ip

Displays network interfaces and IP addresses.

Example:
ip a

---

# ss

Displays active sockets and listening ports.

Example:
ss -tulnp

---

# curl

Sends HTTP requests from terminal.

Example:
curl https://google.com

---

# dig

Performs DNS queries.

Example:
dig google.com
